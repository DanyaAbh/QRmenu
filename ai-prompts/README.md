# AI Prompts For Yanina Menu

This folder contains prompt packs for generating missing dish and cafe drink images for the Yanina menu.

## Scope

- Includes dishes and cafe drinks that still do not have photos in `yanina.html`
- Excludes factory-packaged drinks such as water, cola, juice, and Lipton
- Keeps the "Вода и соки" section out of generation prompts
